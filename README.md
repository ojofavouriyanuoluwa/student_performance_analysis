# student_performance_analysis

## Objectives
To explore the student performance dataset and understand the key variable affecting academic outcome.

## Dataset
This dataset contains information about students' scores, gender, amd test preparation

## Methods
- Data cleaning
- Visualization
- T-test
- Chi-square test

## Key findings
- Students who completed test preparation course performed better
- There is a significant relationship between variables

## Conclusion
The analysis shows that test preparation course significantly influence students performance.
